# Proyecto ZetaOne

Repositorio del proyecto ZetaOne.

Instrucciones rápidas para contribuir:

- Clonar:

```powershell
git clone https://github.com/Evargasz/Proyecto-ZetaOne.git
cd Proyecto-ZetaOne
```

- Crear una rama de trabajo:

```powershell
git switch -c mi-rama
# hacer cambios
git add .
git commit -m "Describir cambios"
git push -u origin mi-rama
```

- Nota sobre binarios grandes:

El instalador grande se movió a Git LFS. Si vas a clonar y trabajar con él, instala `git-lfs` primero:

```powershell
# Windows (PowerShell)
choco install git-lfs -y  # si usas Chocolatey
git lfs install
```
